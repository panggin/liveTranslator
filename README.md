# liveTranslator

SSU AI융합학부 개인 졸업작품

## Intro
OCR(광학 문자 인식) 기술을 활용한 실시간 자막 번역 프로그램입니다.

사용자가 지정한 부분의 영상을 캡처하여 글자를 추출하고 한글로 번역하여 오버레이 화면에 배치합니다.

데모 영상은 [유튜브 링크](https://youtu.be/pzr_sqB-RSk)에서 확인 가능합니다.
